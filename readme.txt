CSVファイルやGoogleスプレッドシートのデータを元に、魅力的なアニメーションバーチャートレースを作成・表示するプラグインです。

== Description ==

Bar Chart Race Pro は、時系列で順位が変動する「バーチャートレース（Bar Chart Race）」を WordPress サイト上で簡単に作成・表示できるプラグインです。
外部の重いライブラリに依存せず、軽量かつ高機能なチャート生成を実現しています。

**主な機能:**

* **多様なデータ入力:**
    * CSVファイルのアップロード（ドラッグ＆ドロップ対応）。
    * Googleスプレッドシートの公開URLからの直接データ読み込み。
    * Shift-JISなどのエンコーディングや、日本語カラム名（日付、ラベル、値など）を自動検知して読み込みます。

* **高度なデザインカスタマイズ:**
    * **カラー設定:** 5種類のプリセットパレットに加え、項目ごとの色を個別に指定可能。全体の文字色も変更できます。
    * **アイコン表示:** 項目の左側やバーの内部に、任意のアイコン画像を表示できます。
    * **背景画像:** チャートの背景に画像を敷くことができ、メディアライブラリから簡単に選択可能です。
    * **表示モード:** ラベルの表示位置（左側・バー内部）ごとに「テキスト」「アイコン」「なし」を柔軟に組み合わせられます。

* **便利な管理・編集機能:**
    * **リアルタイムプレビュー:** 設定を変更すると即座にプレビューに反映されます。
    * **簡易データエディタ:** アップロード後のデータを管理画面上で直接修正・追加・削除できます。
    * **データ同期 (Sync):** Googleスプレッドシートのデータが更新された場合、ワンクリックで最新データを取り込めます。

* **柔軟な再生コントロール:**
    * 再生/一時停止、リセットボタンに加え、シークバーで好きな時点へジャンプできます。
    * ループ再生のON/OFF設定が可能です。

* **ショートコード対応:**
    * `[chart_race id="..."]` : 作成したチャートを記事に埋め込みます。
    * `[chart_race_upload_form]` : ユーザーがフロントエンドからCSVをアップロードしてチャートを作成できるフォームを表示します。

**データの形式:**

以下の3つのカラムを含むCSVデータが必要です。
* `time` (または 日付, 年, 月 など)
* `label` (または 名前, 項目, カテゴリ など)
* `value` (または 数値, 値, 売上 など)

== Installation ==

1.  WordPress管理画面の「プラグイン」>「新規追加」>「プラグインのアップロード」から、本プラグインのzipファイルをアップロードします。
2.  プラグインを有効化します。
3.  管理画面メニューに「Chart Race」が追加されます。

== Frequently Asked Questions ==

= Googleスプレッドシートが読み込めません =
スプレッドシートの共有設定が「リンクを知っている全員」になっていることを確認してください。また、シートのURLを正しく入力しているか確認してください。

= 項目ごとに特定の色を指定したいのですが =
チャート編集画面の「Custom Label Settings」を開くと、データ内の項目名一覧が表示されます。アコーディオンを開いて、それぞれの色やアイコン画像を設定してください。
